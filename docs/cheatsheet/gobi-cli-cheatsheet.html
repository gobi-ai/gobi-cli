Gobi CLI Cheatsheet v2.0.2

Spaces · Global · Vault · Saved · Sessions · Media
Installbrew tap gobi-ai/tap && brew install gobi
npm i -g @gobi-ai/cli
Setupgobi auth login → gobi init → gobi space warp

Auth & Init login & vault setup

  • auth login · auth status · auth logout
  • init · interactive vault selection & .gobi/settings.yaml

Space posts in a space

  • space list · space warp [slug]
  • space create-post --title --content [--auto-attachments]
  • space create-reply <id> --content
  • space feed · list-posts · get-post <id>
  • space list-topics · list-topic-posts <slug>

Vault publish & sync

  • vault publish · upload PUBLISH.md to webdrive
  • vault unpublish
  • vault sync [--dry-run] [--full] [--path]
  • vault sync --conflict ask|server|client|skip
  • vault sync --upload-only | --download-only

Global posts in your vault & public feed

  • global feed [--limit --cursor]
  • global create-post --title --content [--auto-attachments]
  • global create-reply <id> --content
  • global list-posts [--mine] · get-post
  • global edit-post / delete-post <id>

Saved notes & bookmarked posts

  • saved note create --content [--timezone]
  • saved note list [--date YYYY-MM-DD] [--limit]
  • saved note get / edit / delete <id>
  • saved post create --source <id>
  • saved post list [--type all|article|space-post] · get / delete <id>

Session chat with agent on vault knowledge

  • session list [--limit]
  • session get <id> [--limit --cursor]
  • session reply <id> --content

Sense activity & transcripts

  • sense activities --start-time --end-time
  • sense transcriptions --start-time --end-time
  • times are ISO 8601 UTC

Draft agent-suggested actions

  • draft list [--limit] · get <id>
  • draft add <title> <content> [--action --session]
  • draft action <id> <idx> · prioritize <id> <n>
  • draft revise <id> <comment> [--title --content --action]
  • draft delete <id>

Media — Image

  • media image-generate --prompt --aspect-ratio --wait -o
  • media image-edit --prompt --image (img2img)
  • media image-inpaint --prompt --image --mask
  • media image-status / image-download <jobId>

Media — Video & Avatar

  • media cinematic-create --prompt --wait -o
  • media video-create --avatar-id --voice-id --script --wait -o
  • media avatar-from-selfie --image [--prompt]
  • media avatar-design / avatar-confirm
  • media avatars · voices · upload <file>
