Gobi CLI Cheatsheet v2.0.45

Spaces · Personal · Vault · Sense · Artifact · Media
Installbrew tap gobi-ai/tap && brew install gobi
npm i -g @gobi-ai/cli
Setupgobi auth login → gobi vault init → gobi space warp

Auth & Setup login, vault, space

  • auth login · auth status · auth logout
  • vault init · interactive vault selection, writes .gobi/settings.yaml + PUBLISH.md
  • vault list · list vaults you own

Space posts in a space

  • space list · space warp [slug]
  • space create-post --title --content [--artifact --attach --repost-post-id]
  • space create-reply <id> --content [--attach]
  • space feed · list-posts · get-post <id> [--full]
  • space list-topics · list-topic-posts <slug>

Vault admin, publish & sync

  • vault create <slug> --name · rename <new> · delete <slug>
  • vault publish · upload PUBLISH.md to webdrive · unpublish
  • vault status · show publish state & profile URL
  • vault sync [--dry-run] [--full] [--path]
  • vault sync --conflict ask|server|client|skip · --upload-only | --download-only

Personal your private feed

  • personal feed [--limit --cursor]
  • personal create-post --title --content [--artifact --attach --repost-post-id]
  • personal create-reply <id> --content [--attach]
  • personal list-posts · get-post [--full] · edit-post / delete-post
  • private scope — posts and replies visible only to you

Sense activity & transcripts

  • sense list-activities --start-time --end-time
  • sense list-transcriptions --start-time --end-time
  • times are ISO 8601 UTC

Artifact versioned post creations

  • artifact list [--kind --limit] · get <id>
  • artifact create --kind [--file --content --title --vault-slug --post-id --auto-attachments]
  • artifact revise <id> [--file --content --from --change-note]
  • artifact publish <id> --revision · revert <id> --to
  • artifact history <id> · download <id> · delete <id>

Media — Image

  • media generate-image --prompt --aspect-ratio --wait -o
  • media edit-image --prompt --image (img2img)
  • media inpaint-image --prompt --image --mask
  • media get-image-status / download-image <jobId>

Media — Video & Avatar

  • media create-cinematic --prompt --wait -o
  • media create-video --avatar-id --voice-id --script --wait -o
  • media design-avatar-from-selfie --photo [--prompt]
  • media design-avatar / confirm-avatar
  • media list-avatars · list-voices · upload <file>
